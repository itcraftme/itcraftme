# Basic knowledge
  - '.'       Any character except new line
  - '[]'      Matches characters in brackets  
  - '{'    
  - '()'      Group  
  - '[^ ]'    Matches characters not in bracket  
  - '$'       End of a string  
  - '|'    
  - '?'    
  - '*'  
  - '+'  
  - '\d'      Digit (0-9)
  - '\D'      Not digit (0-9)
  - '\w'      Word Character (a-z,A-Z,0-9,_ )
  - '\W'      Not word character
